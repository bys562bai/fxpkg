# 简介
一个包裹管理器

目前主要支持C++，以后考虑支持更多语言

# 定义
包裹: 可以独立编译和使用的最小单元

包裹接口: 用于安装包裹的工具

安装步骤: 包括配置，下载，解压，编译，安装

# 文件夹
port: 接口文件

src: 源文件

install: 安装好的文件，可直接使用

cache: 缓存，放置中间步骤生成的文件

temp: 临时文件，在每次运行结束后自动进行清理

# 目标功能
## 基本
安装，更新，删除，使用自动化，并能自动处理依赖

允许添加手动安装的包裹信息，并纳入包管理中

允许通过函数查询已安装的包裹信息

要使用库时，可以根据要求，自动生成全部或部分的构建脚本


## 包裹协议

作者应当公布自己的数字证书，并对包裹接口进行数字签名
